E-Commerce Website

A fully integrated e-commerce solution built using Spring Boot, Spring Security, OAuth, Thymeleaf, HTML, CSS, and MySQL. This platform offers a seamless shopping experience with robust functionalities, role-based authentication, and an intuitive admin dashboard.

Key Features

  • Login Page: Secure login for users using Spring Security, ensuring authentication and redirection based on user roles.
  • Register Page: Users can register, providing personal details, with input validation and secure data storage in MySQL.
  • Super Admin Page: A page for the Super Admin to manage admin roles and access control.
  • Admin Management: Super Admins can create, edit, and delete admin accounts.
  • Create Admin Page: Super Admins can create new admin accounts with privileges to manage the website.
  • Product Management: Admins can add, edit, and delete products, managing inventory seamlessly.
  • Admin Dashboard: Admins can manage product inventory, user orders, and view detailed statistics on sales.
  • Home Page: A user-friendly home page to browse products, filter by categories, and sort by price.
  • User Profile Page: Users can manage their profiles, view orders, track their shopping cart, and update account settings.

Bank Management System

The Bank Management System is a RESTful API built with Spring Boot, designed for managing bank accounts, transactions, and user authentication. This system allows clients to perform actions such as account creation, deposits, withdrawals, and viewing account details through secure API endpoints.

Tech Stack

Java Spring Boot REST APIs

Features

  • User Account Creation

    User account creation via API with secure validation

  • Deposit and Withdrawal

    Deposit and withdrawal functionality for managing account balances

  • Account Details

    Account details retrieval with the balance and user information

  • Account Management

    Account deletion via API with proper validation

API Endpoints

The system exposes the following RESTful API endpoints:

HTTP Method Endpoint Description
POST /account/create Creates a new bank account with user details.
GET /account/{id} Fetch account details by account number.
GET /account/getallaccount Get a list of all bank accounts.
PUT /account/deposit/{accountNumber}/{amount} Deposit an amount into an existing account.
PUT /account/withdraw/{accountNumber}/{amount} Withdraw an amount from an existing account.
DELETE /account/delete/{accountNumber} Delete an existing account.

Chat Application

A real-time messaging platform built using Spring Boot and WebSocket for instant communication. The application supports a seamless user experience, allowing users to send and receive messages in real-time. With WebSocket integration, it offers instant messaging functionality, ensuring a responsive chat environment.

Key Features

  • Real-time Messaging: Send and receive messages instantly using WebSocket and STOMP.
  • User Authentication: Users can input their names to join the chat room.
  • Broadcasting: Messages are broadcasted to all connected clients in the chat room.
  • Responsive Design: Built with Bootstrap for a responsive, user-friendly interface across all devices.
  • Simple Interface: The app provides an intuitive UI for seamless user experience.

Tech Stack

Java Spring Boot WebSocket JavaScript Bootstrap

Technologies Used

The project utilizes Spring Boot to handle the backend, implementing WebSocket for real-time communication. On the front-end, Bootstrap ensures a responsive design, while JavaScript is used to handle the interaction between the client and server.
